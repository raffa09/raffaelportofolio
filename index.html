My Skills


HTML

HTML adalah bahasa standar untuk membuat struktur dan konten halaman web menggunakan tag-tag seperti (html), (p), dan (a), serta dapat dikombinasikan dengan CSS dan JavaScript untuk tampilan dan interaktivitas.

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan gaya elemen HTML di halaman web. CSS mengatur warna, ukuran, layout, font, dan efek visual lainnya agar web lebih menarik dan responsif.

JS

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan logika pada halaman web, seperti membuat tombol, animasi, validasi form, atau memanipulasi elemen HTML secara dinamis. JavaScript bekerja di sisi klien (browser) dan membuat web lebih hidup serta responsif
